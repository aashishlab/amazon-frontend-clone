# 🛒 Amazon UI Clone

A fully responsive Amazon UI clone built using modern frontend technologies.  
This project replicates the design and layout of Amazon’s homepage to practice UI/UX and frontend development skills.

## 🚀 Features
- Responsive design for all screen sizes
- Navigation bar similar to Amazon
- Product sections and cards
- Clean and structured layout
- Interactive UI elements

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript / React (if used)

## 📌 Purpose
This project was built for learning and improving frontend development skills by recreating a real-world application.
